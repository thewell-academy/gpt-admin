HTML로 수학 공식 작성하기

이 가이드는 HTML과 MathML을 사용하여 웹 페이지에서 수학 공식을 작성하는 방법을 설명합니다.

1. 기본 HTML로 간단한 수식 작성

HTML에서 간단한 수학 기호는 유니코드 문자를 사용하여 표현할 수 있습니다:

  • 덧셈: 3 + 2 = 5
  • 뺄셈: 7 − 4 = 3
  • 곱셈: 6 × 3 = 18
  • 나눗셈: 10 ÷ 2 = 5

2. MathML로 복잡한 수학 공식 작성

MathML은 HTML 문서에서 수학 공식을 작성하기 위한 표준입니다. 아래는 예제입니다:

2.1 간단한 분수

다음은 분수를 표현하는 MathML 코드입니다:

                <math xmlns="http://www.w3.org/1998/Math/MathML">
                    <mfrac>
                        <mn>1</mn>
                        <mn>2</mn>
                    </mfrac>
                </math>

결과:

1 2

2.2 루트 표현

제곱근을 표현하는 MathML 코드:

                <math xmlns="http://www.w3.org/1998/Math/MathML">
                    <msqrt>
                        <mn>16</mn>
                    </msqrt>
                </math>

결과:

16

2.3 이중 지수

다음은 지수 표현입니다:

                <math xmlns="http://www.w3.org/1998/Math/MathML">
                    <msup>
                        <mn>2</mn>
                        <mn>3</mn>
                    </msup>
                </math>

결과:

2 3

3. 응용 예제

수능 시험에서 볼 수 있는 수식 예제를 MathML로 작성해보겠습니다:

문제: 다음 식의 값을 구하시오: \( \frac{3x}{2} + 5 \)

                <math xmlns="http://www.w3.org/1998/Math/MathML">
                    <mfrac>
                        <mn>3x</mn>
                        <mn>2</mn>
                    </mfrac>
                    <mo>+</mo>
                    <mn>5</mn>
                </math>

결과:

3x 2 + 5

1. 기본 수학 연산 기호

MathML을 사용하여 기본 수학 연산 기호를 표현할 수 있습니다. 아래는 예제입니다:

1.1 덧셈, 뺄셈, 곱셈

  • 덧셈: 3 + 2는 직접 텍스트로 표현 가능
  • 뺄셈: 7 − 4는 텍스트로도 표현 가능
  • 곱셈: 6 × 3

1.2 나눗셈

나눗셈 기호(÷)는 MathML의 <mo> 태그로 표현할 수 있습니다:

                <mrow>
                    <mn>10</mn>
                    <mo>÷</mo>
                    <mn>2</mn>
                </mrow>

결과:

10 ÷ 2

2. 극한(Limit) 표현

극한은 MathML의 <munder> 태그를 사용하여 표현합니다. 아래는 예제입니다:

                <mrow>
                    <mo>lim</mo>
                    <munder>
                        <mi>x</mi>
                        <mo>→</mo>
                        <mn>0</mn>
                    </munder>
                    <mo>(sin</mo>
                    <mi>x</mi>
                    <mo>/</mo>
                    <mi>x</mi>
                    <mo>)</mo>
                </mrow>

결과:

lim x → 0 (sin x / x )

3. 적분(Integral) 표현

적분 기호는 MathML의 <msubsup> 태그를 사용합니다. 아래는 예제입니다:

3.1 정적분

                <mrow>
                    <msubsup>
                        <mo>∫</mo>
                        <mn>0</mn>
                        <mn>1</mn>
                    </msubsup>
                    <mi>x</mi>
                    <mo>²</mo>
                    <mo>dx</mo>
                </mrow>

결과:

∫ 0 1 x ² dx

3.2 무한 적분

                <mrow>
                    <msubsup>
                        <mo>∫</mo>
                        <mn>-∞</mn>
                        <mn>+∞</mn>
                    </msubsup>
                    <mi>e</mi>
                    <mo>^</mo>
                    <mo>-x²</mo>
                    <mo>dx</mo>
                </mrow>

결과:

∫ -∞ +∞ e ^ -x² dx

1. 수학 기호 작성하기

MathML을 사용하면 다양한 수학 기호를 표현할 수 있습니다.

1.1 π (파이)

π 기호를 표현하려면 아래와 같은 MathML 코드를 사용합니다:

            <mi>π</mi>

결과:

π

1.2 θ (세타)

θ 기호를 표현하려면 아래 코드를 사용합니다:

            <mi>θ</mi>

결과:

θ

2. 분수와 나눗셈

MathML로 분수 및 나눗셈 기호를 표현할 수 있습니다.

2.1 분수

분수를 표현하는 기본 코드:

            <mfrac>
                <mn>1</mn>
                <mn>2</mn>
            </mfrac>

결과:

1 2

2.2 나눗셈 기호 (÷)

나눗셈 기호는 다음과 같이 작성합니다:

            <mo>÷</mo>

예제:

            <mrow>
                <mn>10</mn>
                <mo>÷</mo>
                <mn>2</mn>
            </mrow>

결과:

10 ÷ 2

3. 복잡한 표현식

3.1 케이스 구분식

아래 코드는 케이스 구분을 표현하는 예제입니다:

            <mrow>
                <mo>f(x) = </mo>
                <mo>{</mo>
                <mtable columnalign="left">
                    <mtr>
                        <mtd><mn>5x + a</mn></mtd>
                        <mtd><mo>(x < -2)</mo></mtd>
                    </mtr>
                    <mtr>
                        <mtd><mn>x² - a</mn></mtd>
                        <mtd><mo>(x ≥ -2)</mo></mtd>
                    </mtr>
                </mtable>
            </mrow>

결과:

f(x) = { 5x + a (x < -2) x² - a (x ≥ -2)
